# 🎵 Spotify Hit Song Prediction (Machine Learning Project)

## 📌 Project Overview

This project focuses on predicting whether a **Spotify song will become a Hit or Not Hit** using **Machine Learning** techniques based on various musical audio features.

The project follows a complete **Machine Learning pipeline**, including data preprocessing, feature engineering, model training, evaluation, and deployment, to identify the characteristics of successful songs.

## 🎯 Project Goal

* 🧹 Clean and preprocess raw Spotify data
* 📊 Perform Exploratory Data Analysis (EDA)
* 🎵 Analyze musical features influencing song popularity
* 🤖 Build and compare multiple Machine Learning models
* 📈 Predict whether a song is likely to become a hit

---

## ❗ Problem Statement

With millions of songs released every year, predicting a song's success has become a major challenge for artists, producers, and music streaming platforms.

Song popularity depends on multiple audio characteristics such as:

* 🎶 Danceability
* ⚡ Energy
* 🎤 Speechiness
* 🎧 Acousticness
* 🎼 Tempo
* 🔊 Loudness
* ❤️ Valence
* 🎹 Instrumentalness

This project leverages Machine Learning to analyze these features and predict whether a song has the potential to become a **Hit Song**.
